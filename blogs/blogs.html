Q1: What is JavaScript?

Answer: JavaScript is a versatile programming language primarily used to create and control dynamic website content. It allows for the implementation of complex features on web pages, such as interactive forms, animations, and more.

Q2: How do you include JavaScript in an HTML file?

Answer: JavaScript can be included in an HTML file using the <script> tag. It can be embedded directly within the HTML file or linked as an external file. For example: 

<script src="script.js"></script>

Q3: What is the DOM in JavaScript?

Answer: The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. JavaScript can interact with the DOM to dynamically update the content and structure of a web page.

Q4: What is an event in JavaScript?

Answer: An event in JavaScript refers to actions or occurrences that happen in the system being programmed. Examples of events include clicking a button, submitting a form, or loading a webpage. JavaScript can react to these events using event listeners.

Q5: How do you declare a variable in JavaScript?

Answer: Variables in JavaScript can be declared using the var, let, or const keywords. For example: 

var name = "John";
let age = 30;
const isStudent = true;

var is function-scoped, while let and const are block-scoped. const is used for variables that should not be reassigned.
